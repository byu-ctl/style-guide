Forms

Every page in Learning Suite (instructor side) is wrapped in a form element. That means form controls work anywhere.

The controls below are all created using LS control objects in PHP. You don't usually create them directly in the HTML. Refer to the LS developer wiki for documentation.

Basic Elements

Checkboxes

Checkboxes are created with the CheckBox control. Labels can be aligned either to the right or left.

Output:

<div class="inlineLabel ">
  <input type="checkbox" id="cb" class="" name="cb">
  <label id="cb_label" for="cb" class="">Checkbox 1</label>
</div>
<div class="inlineLabel ">
  <label id="cb_label" for="cb" class=" ">Checkbox 1</label>
  <input type="checkbox" id="cb" class="" name="cb">
</div>
